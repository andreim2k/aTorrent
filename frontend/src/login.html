Google Sign-In Required

Configure GOOGLE_CLIENT_ID in backend/.env to enable authentication

Setup:

  1. Get Client ID from Google Cloud Console
  2. Add to backend/.env as GOOGLE_CLIENT_ID
  3. Replace YOUR_GOOGLE_CLIENT_ID on line 189
  4. Restart services with ./app.sh restart
